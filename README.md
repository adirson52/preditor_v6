# Preditor FCU - Base 31 completa

Site estatico do Projeto Revelando Favelas para publicacao dos dashboards
da Base 31 completa, com dez areas de estudo.

## Arquivos principais

- `index.html`: pagina inicial.
- `07_dashboard_ebm_avancado_areas_estudo_completo.html`: modelo completo.
- `07_dashboard_ebm_avancado_areas_estudo_morfologico.html`: modelo morfologico.
- `08_dashboard_metricas_modelos_areas_estudo.html`: analise dos modelos.
- `data_tiles/`: pontos do mapa em tiles comprimidos e poligonos FCU.
- `31_nomenclatura_variaveis_site.txt`: nomes e descricoes das variaveis exibidas.

## Publicacao

O projeto e um site estatico publicado a partir do repositorio
`adirson52/preditor_v6`, com integracao automatica entre GitHub e Vercel.
A raiz de publicacao e esta pasta.
